We've all been there — your essay is well-written and covers the topic thoroughly, but you're half a page short of the requirement. Before you start adding filler sentences, consider that your font choice can make a significant difference in page count.

## How Fonts Affect Page Length

Not all fonts are created equal. Some fonts have wider characters, taller line heights, or more generous spacing between letters. These differences can add up to a noticeable change in page count, even with the same word count.

Here's how common fonts compare at 12pt, double-spaced, on letter paper with 1-inch margins:

| Font | Words Per Page | Pages for 1,000 Words |
|------|---------------|----------------------|
| Courier New | ~210 | ~4.8 |
| Georgia | ~230 | ~4.3 |
| Times New Roman | ~250 | ~4.0 |
| Palatino | ~245 | ~4.1 |
| Arial | ~225 | ~4.4 |
| Calibri | ~240 | ~4.2 |
| Verdana | ~195 | ~5.1 |

## Top Fonts That Take Up More Space

### 1. Verdana
Verdana was designed for screen readability, which means it has wider characters and generous spacing. It produces roughly **20% more pages** than Times New Roman at the same size.

### 2. Courier New
As a monospaced font, every character in Courier New takes up the same width. This makes it one of the most space-consuming fonts available. However, it looks obviously different from standard essay fonts, so use it only if your guidelines allow it.

### 3. Georgia
Georgia is a serif font similar to Times New Roman but slightly wider. It's a subtle swap that most professors won't notice but can add 10–15% more pages.

### 4. Book Antiqua
Similar to Palatino, Book Antiqua has elegant, slightly wider characters that take up more space while maintaining a professional, academic look.

## Other Legitimate Ways to Add Length

- **Adjust font size slightly** — 12.5pt instead of 12pt is barely noticeable but adds space
- **Use 2.1 or 2.15 line spacing** — Slightly more than double-spaced, hard to detect
- **Expand your margins slightly** — 1.1-inch margins instead of 1 inch
- **Add section headings** — They break up text and add visual length
- **Include a header with your name, date, and course** — Standard academic formatting that adds lines

## What NOT to Do

- Don't use an obviously inappropriate font (Comic Sans for an academic paper)
- Don't increase the size of periods and commas — professors know this trick
- Don't add extra spaces between words — easily detectable with "Show Formatting" in Word
- Don't use size 13 font when 12 is required — this is easily caught
